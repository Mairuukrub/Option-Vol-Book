Webull Thailand — ใช้งานจริง

หน้า option chain บน Webull แสดง Greeks ทุกตัวสำหรับแต่ละ strike/expiry คอลัมน์ที่ควรดูทุกครั้งที่เปิด position: Delta, Theta, Vega, IV%

ก่อนกด Buy/Sell ลอง "อ่าน Greeks" ให้เป็นนิสัย — เช่น ถ้า Theta = -0.08 แปลว่าถ้าถือข้ามคืน option หายไป ~8 cents/วัน โดยอัตโนมัติ

3 วิธีตีความ Delta

① Rate of Change
Call delta 50 = option ขยับ $0.50 ต่อทุก $1 ที่หุ้นขึ้น/ลง

Call delta 25 = option ขยับ $0.25 ต่อ $1
② Hedge Ratio
จำนวนหุ้นที่ต้อง hedge ต่อ 1 option
Hedge Ratio = 100 / Delta

ATM call (Δ50): ต้องใช้หุ้น 2 หุ้น/option (100/50)
③ ITM Probability
|Delta| ≈ โอกาสที่ option จะ expire ITM

Delta 25 → ~25% chance ว่าจะ ITM ตอน expire
💡 Key Insight — Delta เปลี่ยนตามเวลาและ IV
เมื่อเวลาลดลง (ใกล้ expiry) หรือ volatility ลดลง → delta ของ ITM calls เคลื่อนไป 100, delta ของ OTM calls เคลื่อนไป 0

เมื่อเวลาเพิ่มขึ้น (LEAPS) หรือ volatility สูง → delta ทุก strike เคลื่อนเข้าหา 50 (options ดู ATM หมด)

สูตรง่าย: ↑IV / ↑Time → delta pulls to 50  |  ↓IV / ↓Time → delta pushes to extremes
⚠ Delta เป็น approximation เท่านั้น
Delta บอก direction risk ณ จุดเวลานั้น แต่ไม่ใช่ตัวเดิมตลอด เพราะ Gamma ทำให้ delta เปลี่ยนตลอดเวลาที่ราคาขยับ นักเทรดที่ดูแค่ delta แล้วคิดว่า "delta neutral = ปลอดภัย" อาจได้รับความเสียหายหนักจาก gamma
Webull — ใช้ Delta ในการเทรด

CSP (Cash Secured Put): เลือก strike ที่มี delta ราว 20–30 → โอกาส expire worthless ~70–80% ทฤษฎี เหมาะกับ income strategy

Covered Call: ถ้าขาย call delta 30 แปลว่า option มีโอกาส ~30% ที่จะถูก exercise — ดูจาก Greeks ใน option chain ของ Webull ได้เลย

LEAPS UUUU: LEAPS 12+ เดือน ที่ delta ~60–70 เปรียบเสมือนถือหุ้น 60–70% โดยจ่าย premium น้อยกว่า — นี่คือ "synthetic position" แบบหนึ่ง

The Gamma

RATE OF CHANGE IN DELTA (CURVATURE)
pp. 103–110 · อัตราที่ delta เปลี่ยนเมื่อ underlying ขยับ
Γ = ΔDelta / ΔUnderlying Price ถ้า option มี gamma 5 → ทุก $1 ที่ underlying ขึ้น delta เพิ่ม 5 | ทุก $1 ที่ลง delta ลด 5
ข้อเท็จจริงสำคัญเรื่อง Gamma
• ทั้ง Call และ Put มี gamma บวก เสมอ
• Long option = Long gamma
• Short option = Short gamma
• ATM options มี gamma สูงที่สุด
• Underlying contract มี gamma = 0
Gamma กับ Moneyness
ATM → Gamma สูงสุด (bell curve peak)
ITM / OTM → Gamma ลดลงทั้งสองข้าง
ใกล้ expiry → ATM gamma พุ่งสูงมาก
ห่าง expiry → gamma เรียบ flat กว่า
🔑 Gamma ทำให้ Delta Neutral ล้มเหลวได้
สมมติขาย call 10 สัญญา (delta -300 รวม) → คิดว่า delta neutral
แต่ถ้า gamma แต่ละ call = 6 → ทุก $1 ที่ขึ้น delta เพิ่ม 60 per contract
ราคาขึ้น 10 จุด → delta รวมกลายเป็น -300 + (6×10×10) = +300
จาก delta neutral กลายเป็น long 300 deltas โดยไม่ได้ตั้งใจ!

📖 Historical Event — COMEX Collapse 1985
กลุ่มนักเทรดขาย OTM gold options จำนวนมาก (short gamma position ขนาดใหญ่) ช่วงก่อนหน้าตลาดทองเงียบ → กลยุทธ์นี้ทำกำไรดี แต่เมื่อตลาดทองพุ่งขึ้นแรง delta ของ position เพิ่มเป็นเท่าตัวอย่างรวดเร็ว ขาดทุนมหาศาล → clearing firm ล้มละลาย → วิกฤติ COMEX clearing association บทเรียน: short gamma = ความเสี่ยงระเบิดได้ ต้องมี risk limit ที่ชัดเจน
Webull — Gamma ในบริบท income strategy

Gamma risk สำหรับ CSP/Covered Call: short options ทุกตัวมี short gamma ถ้า expiry ใกล้มาก (weekly options) gamma จะสูง → หุ้นขยับนิดเดียว delta เปลี่ยนเยอะ → P&L กระโดด

วิธีจัดการ: ถ้าไม่ต้องการ gamma risk สูง หลีกเลี่ยง expiry ที่สั้นเกินไป Natenberg แนะให้ดู gamma ของ position รวม ไม่ใช่แค่ position เดียว

TE Collar case: ตอน TE ลงแรง 19% — นั่นคือ gamma risk กำลังทำงาน CSP ที่เคย delta 25 กลายเป็น delta 70+ ในชั่วข้ามคืน เพราะ gamma ดัน delta ให้วิ่ง

ลักษณะของ Theta

ATM Theta สูงสุด
ATM options เสีย time value เร็วที่สุด (dollar terms)
ITM/OTM เสียช้ากว่า
Theta เร่งใกล้ Expiry
ยิ่งใกล้ expiration ATM theta เร่งสูงขึ้น ช่วง last week theta เยอะมากเป็นพิเศษ
IV สูง → Theta สูง
IV สูง = time value เยอะ → ทุกวันที่ผ่านไป theta ดึงมากขึ้น ขาย option ตอน IV สูง ได้ theta เยอะกว่า
💡 Theta กับ IV — เชื่อมกันลึกกว่าที่คิด
IV สูง = มี time value เยอะ = theta/วันก็สูงด้วย (เพราะมีอะไรให้ decay มากกว่า)
IV ต่ำ = time value น้อย = theta/วันก็ต่ำ

นั่นคือ CSP/CC ที่ขายตอน IV สูง (เช่น ก่อน earnings หรือช่วง volatile) จะได้ theta มากกว่า = income strategy ทำงานได้ดีกว่า
Webull — Theta คือ "เงินเดือน" ของ options seller

CSP/CC strategy: ทุกวันที่ position ถือ option seller ได้ theta เป็นกำไร เช่น sell CSP มี theta +0.05 → ได้ $5/วัน/contract โดยไม่ต้องทำอะไร

สูตรประมาณ break-even: ถ้า underlying ไม่ขยับ theta วิ่งไปเรื่อยๆ 45 DTE CSP ที่ sell ราคา $1.50 → ถ้า theta เฉลี่ย $0.03/วัน → 50 วัน expire worthless เก็บเต็ม

⚠ Webull แสดง Theta ต่อวัน ตรวจสอบก่อน บาง platform แสดง theta/วัน บางอันแสดง/7วัน อย่าสับสน

ν

The Vega (Kappa)

VOLATILITY SENSITIVITY
pp. 113–116 · มูลค่าที่เปลี่ยนต่อ 1% ที่ IV ขยับ
Vega = ΔOption Value / Δ1% Implied Volatility Natenberg ใช้ชื่อ "kappa" (κ) แต่ในตลาดเรียก vega ทั่วไป
Vega บวกสำหรับ options ทุกประเภท (calls และ puts)
Vega: บวกเสมอ
IV เพิ่ม → option (ทั้ง call และ put) มีค่าสูงขึ้น
IV ลด → option มีค่าต่ำลง

Long option = Long Vega (ชอบ IV สูง)
Short option = Short Vega (ชอบ IV ต่ำ)
ตัวอย่าง
Call มีค่า $3.25 ที่ IV = 20%
Vega = 0.15

ถ้า IV เพิ่มเป็น 21% → call = $3.40
ถ้า IV ลดเป็น 19% → call = $3.10

ลักษณะของ Vega

ATM = Vega สูงสุด ($ terms)
ATM options ไวต่อ IV เปลี่ยนมากที่สุดในแง่ dollar
OTM/ITM vega ต่ำกว่า
OTM = Vega สูงสุด (% terms)
OTM options ที่ถูกกว่า มี vega สูงมากเมื่อเทียบกับราคา (% change ใหญ่กว่า)
Long-term = Vega สูงกว่า
ยิ่ง DTE เยอะ IV มีเวลาส่งผลมากกว่า → LEAPS vega สูงมาก
🔑 Time กับ Volatility มีผลคล้ายกัน
Natenberg เน้นว่า: "Decreasing volatility is similar to decreasing time to expiration"

กราฟของ option value vs. time และ option value vs. volatility มีรูปร่างคล้ายกันมาก → ถ้าจำไม่ได้ว่า IV ลดทำอะไรกับ option ให้นึกถึงว่าเวลาผ่านไปทำอะไร → ผลเดียวกัน

→ IV ลด 1% = เหมือนเวลาผ่านไปหลายวัน (ขึ้นอยู่กับ vega/theta ratio)
⚠ IV Crush — ปัญหาใหญ่สำหรับ Long Option
ซื้อ call ก่อน earnings → IV = 80% → หลัง earnings ประกาศดีมาก หุ้นขึ้น 5% แต่ IV collapse เหลือ 30% → vega ดึงราคา option ลงมากกว่า delta ดันขึ้น ผลสุดท้าย: call กลับขาดทุน ทั้งที่ทิศทางถูก นี่คือเหตุผลที่เนื้อหา Episode 1 ของซีรีส์ content ที่วางแผนไว้นั้นสำคัญมาก
Webull — ใช้ Vega วิเคราะห์ IV Rank

IV Rank (IVR) บน Webull: ถ้า IVR สูง (>50%) → IV สูงกว่าปกติ → options แพงในแง่ vega → เป็นโอกาส sell options (short vega) เก็บ premium แพง

UUUU LEAPS strategy: IVR ต่ำ ~5% → ซื้อ LEAPS ราคาถูก (IV/vega ต่ำ) ถ้า IV กลับมาสูงขึ้นด้วย catalyst → vega ดัน option value ขึ้นได้โดยไม่ต้องรอหุ้นขึ้น

Collar strategy: Long stock + Long put + Short call → vega ของ long put ชดเชย short call บางส่วน → ทำให้ vega exposure ลดลงจากการ hedge ทั้งสองทาง

🧮 Implied Greeks — แนวคิดขั้นสูง
Greeks ที่คำนวณจาก implied volatility (ราคาตลาด) แทนที่จะเป็น future volatility ที่เราเดา

Implied Delta: บอกว่าที่ IV ปัจจุบัน delta คือเท่าไหร่ — traders นิยมใช้เพื่อ maintain delta neutral position ที่สอดคล้องกับมุมมองตลาดจริง

Implied Theta: เช่นเดียวกัน — theta ที่คำนวณจาก IV ปัจจุบัน มักแสดงบน trading platforms

เวลา roll หรือ adjust: คำนวณ Greeks รวมของ position ทั้งหมด (net delta, net theta) ไม่ใช่แค่ option ใหม่ที่กำลังเพิ่ม

Rule of thumb: income seller (CSP/CC) ต้องการ theta บวก + vega ลบ + gamma ลบ → ต้องขาย option ตอน IV สูงกว่าปกติ (IVR > 30%)
