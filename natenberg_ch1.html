Natenberg · Option Volatility & Pricing · 1994
CHAPTER 01
The Language of Options
สรุปเนื้อหา · ภาษาพื้นฐานของ Options
TL;DR

บทนี้ปูพื้นคำศัพท์ที่ต้องรู้ก่อนเข้าสู่เนื้อหาลึก — ตั้งแต่ anatomy ของ option contract, Intrinsic vs Time Value, ITM/OTM/ATM, ไปจนถึงระบบ settlement และ margin ที่ส่งผลต่อ cash flow จริงในการเทรด

01 Contract Specifications · ส่วนประกอบของ Option
Underlying

สินทรัพย์อ้างอิง เช่น หุ้น, futures, index ที่ option นั้น track อยู่

Exercise Price (Strike)

ราคาที่สิทธิ์ในการซื้อหรือขายจะถูก execute เมื่อ exercise

Expiration Date

วันสิ้นสุดอายุ option หลังจากนี้สิทธิ์จะหมดทันที

Type: Call / Put

Call = สิทธิ์ซื้อ · Put = สิทธิ์ขาย underlying ที่ strike price

American vs European

American — exercise ได้ทุกวันก่อน expiry (หุ้น US ส่วนใหญ่) · European — exercise ได้เฉพาะวัน expiry เท่านั้น (index options บางตัว เช่น SPX)

02 Premium = Intrinsic Value + Time Value

ราคา option ที่เห็นใน chain ประกอบด้วย 2 ส่วนเสมอ:

Premium = Intrinsic Value + Time Value
ทั้งสองรวมกันต้องเท่ากับ premium เสมอ ขาดไม่ได้
Intrinsic Value มูลค่าที่ได้จริงถ้า exercise ทันที — เท่ากับ (S − K) สำหรับ call หรือ (K − S) สำหรับ put ถ้าผลลบ จะเป็น 0 เสมอ ไม่ติดลบ
Time Value มูลค่าส่วนเกินจาก intrinsic ที่ตลาดจ่ายให้เพราะ "ยังมีเวลาเหลือ" — ค่อยๆ หายไปเรื่อยๆ จนเป็น 0 ที่ expiry (Theta decay)
At Parity สถานะที่ time value = 0 คือราคา option เท่ากับ intrinsic value พอดี มักเกิดใกล้ expiry หรือ deep ITM
ตัวอย่าง

หุ้น $62, Put strike $70 → Intrinsic = $8 / ถ้า put ราคา $9 → Time value = $1 เท่านั้น
ทอง $435/oz, Call $400 → Intrinsic = $35 / ถ้า call ราคา $50 → Time value = $15

03 Moneyness · ITM / ATM / OTM

ตำแหน่งของ Strike Price เทียบกับราคา Underlying ปัจจุบัน

CALL

Strike
◀ ITM (Strike < Price) OTM (Strike > Price) ▶

PUT

Strike
◀ OTM (Strike < Price) ITM (Strike > Price) ▶
ITM In-the-Money — มี Intrinsic Value อยู่ · Call ITM เมื่อ Strike < ราคา · Put ITM เมื่อ Strike > ราคา
ATM At-the-Money — Strike ≈ ราคาปัจจุบัน · มี Time Value มากที่สุด · Theta decay เร็วที่สุด
OTM Out-of-the-Money — ไม่มี Intrinsic Value · ราคา = Time Value ล้วนๆ · เป็น 0 ถ้า expire OTM
⚑ สำคัญ

Call ITM → Put เดียวกัน (strike เดียวกัน) ต้อง OTM เสมอ ไม่มีทางเป็น ITM ทั้งคู่พร้อมกัน

04 Exercise & Assignment
Exercise (ผู้ซื้อ)

ผู้ถือ long option ใช้สิทธิ์ → ได้รับ underlying ที่ strike price (call) หรือ ขาย underlying ที่ strike price (put)

Assignment (ผู้ขาย)

ผู้ขาย option ถูก assign โดย clearing house แบบสุ่ม → ต้องส่ง/รับ underlying ตามเงื่อนไข

กระบวนการ: ผู้ซื้อแจ้ง clearing firm → clearing house สุ่มเลือกผู้ขาย → ผู้ขายถูก assign และต้องปฏิบัติตามสัญญา เมื่อ exercise แล้ว option นั้นหยุดมีอยู่ทันที

⚡ สำคัญสำหรับ Covered Call / CSP

ถ้าขาย Covered Call แล้ว option ถูก exercise → หุ้นจะถูกเรียกไปที่ strike price ทันที ไม่มีสิทธิ์เลือก ต้องเตรียมรับสถานการณ์นี้ไว้เสมอเมื่อ short call เข้า ITM

05 Market Integrity & Margin

ระบบ Clearing House ทำให้ options market น่าเชื่อถือ — ไม่ต้องกังวลว่าคู่สัญญาจะ default เพราะ clearing house เป็นผู้ค้ำประกันทุก trade

Margin Call เงินประกันที่ clearing house เรียกเพิ่มเมื่อ position ขาดทุน — สามารถใช้หลักทรัพย์แทนเงินสดได้
Variation Call เรียกเพื่อชำระ ขาดทุนที่เกิดขึ้นแล้ว — ต้องจ่ายเป็น cash เท่านั้น ไม่สามารถใช้หลักทรัพย์แทนได้
⚑ Cash Flow Risk

Webull Thailand ใช้ stock-type settlement — กำไรขาดทุนไม่ realize จนกว่าจะปิด position ถ้า position ขาดทุนหนักอาจเจอ margin call ก่อนที่ราคาจะกลับมาได้

06 Settlement Types
Stock-Type Settlement

จ่ายเต็มจำนวนทันที · กำไร/ขาดทุนเป็น unrealized จนกว่าจะปิด position · ใช้กับ options US ทุกตัว

Futures-Type Settlement

ไม่จ่ายเงินล่วงหน้า · Mark-to-market ทุกวัน · กำไร/ขาดทุน realize ทันทีทุกวัน · ใช้กับ futures

07 Long vs Short ในบริบท Options
Long Position ได้กำไรเมื่อราคา underlying ขึ้น — ไม่ว่าจะเป็น long stock, long call, short put (ล้วนเป็น bullish bias)
Short Position ได้กำไรเมื่อราคา underlying ลง — short stock, long put, short call (ล้วนเป็น bearish bias)
ระวังความสับสน

"Long call" = bullish · "Short call" = bearish หรือ neutral · "Long put" = bearish · "Short put (CSP)" = bullish ต้องระวังไม่งงกับทิศทาง

⚡ CONNECT TO YOUR TRADING

CSP บน Webull: ตอนขาย Cash Secured Put → คือ short put = long bias ถ้า expire OTM ได้ premium เต็ม, ถ้า ITM จะถูก assign หุ้นที่ strike price (ต้องมีเงินสดวาง)

Covered Call บน TE: short call = neutral/bearish bias บน upside ถ้าหุ้นขึ้นเกิน strike จะถูก exercise และหุ้นถูกเรียกออกไป — นี่คือสิ่งที่เกิดขึ้นในสถานการณ์ KORE Power acquisition

Time Value Decay: เมื่อ sell options (CSP/CC) เราได้กำไรจาก time value ที่ค่อยๆ สลายตัว — ยิ่งใกล้ expiry ยิ่งเร็ว (Theta ในบท 6)

✓ คำศัพท์ที่ต้องจำจากบทนี้
  • Premium = Intrinsic Value + Time Value เสมอ
  • ITM Call = Strike < ราคาปัจจุบัน | ITM Put = Strike > ราคาปัจจุบัน
  • ATM = มี time value มากที่สุด, theta decay เร็วที่สุด
  • Exercise = ผู้ซื้อใช้สิทธิ์ | Assignment = ผู้ขายถูกเรียก
  • American option = exercise ได้ก่อน expiry (หุ้น US ทั้งหมด)
  • Intrinsic value ≥ 0 เสมอ ไม่มีติดลบ
  • At Parity = time value = 0, premium = intrinsic value
