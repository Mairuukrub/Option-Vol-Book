Natenberg · Option Volatility & Pricing · 1994
CHAPTER 05
Using an Option's Theoretical Value
สรุปเนื้อหา · เอา Theoretical Value ไปทำกำไรจริงด้วย Delta Hedge
TL;DR

บทนี้แสดงวิธีเปลี่ยน theoretical edge เป็นกำไรจริง ผ่าน 3 ขั้นตอน: ซื้อ option ที่ underpriced → สร้าง delta-neutral hedge → adjust ให้ neutral เรื่อยๆ · ตัวอย่างจริงในหนังสือพิสูจน์ว่า model ทำนายกำไร $63 และผลจริงได้ $62.81 — แม่นมาก ถ้า input ถูก

01 เป้าหมาย · เปลี่ยน Edge เป็นเงินจริง

"If we make these assumptions, and the Black-Scholes Model does indeed work, then we ought to be able to turn any difference between an option's price and its theoretical value into a profit."

— Natenberg, Chapter 5
ตัวอย่างหลักของบทนี้

June 100 call · Theoretical Value = 3.88 · ราคาตลาด = 3.25
→ option underpriced อยู่ 0.63 → ซื้อ เพราะมี theoretical edge

แต่การซื้อแล้วรอเฉยๆ ไม่พอ — Chapter 3 บอกแล้วว่าต้อง hedge ด้วย underlying เพื่อให้ position เป็น delta neutral (ไม่ขึ้นกับทิศทางราคา) แล้วเก็บกำไรจาก edge อย่างเดียว

02 3 ขั้นตอนการใช้ Theoretical Value
1
Purchase (sell) undervalued (overvalued) options
ซื้อ option ที่ราคา < theoretical value หรือขายที่ราคา > theoretical value — นี่คือจุดที่ edge เกิด
2
Establish a delta neutral hedge
hedge ด้วย underlying เพื่อตัด directional risk · ตัวอย่าง: ซื้อ 100 calls (delta 57) → ขาย 57 futures เพื่อให้ delta position = 0
3
Adjust the hedge at regular intervals to remain delta neutral
เมื่อราคาขยับ delta เปลี่ยน → ต้อง rebalance ให้ neutral อยู่เสมอ · ในทางทฤษฎี adjust ต่อเนื่อง แต่จริงๆ adjust เป็นช่วงๆ ได้
Delta คืออะไร (เกริ่นก่อนบท 6)

Delta = hedge ratio · Call delta อยู่ระหว่าง 0–100 · Underlying มี delta = 100 เสมอ · ถ้า call delta = 57 → ซื้อ 100 calls ต้องขาย 57 futures เพื่อ neutral

03 ตัวอย่างจริง · 10 สัปดาห์ของ Delta Hedge

หนังสือเดิน hedge นี้ไปจนถึง expiry โดย adjust ทุกสัปดาห์ตามที่ delta เปลี่ยน — ผลลัพธ์แยกเป็น 4 ส่วน:

COMPONENTP&Lคำอธิบาย
Original Hedge−138.83option + futures ที่เปิดตอนแรก ขาดทุน
Adjustments+205.27กำไรจากการ buy low / sell high ตอน rebalance
Option Carrying Cost−4.99ดอกเบี้ยจากการจ่ายเงินซื้อ option

TOTAL PROFIT+62.81ผลรวมจริง
💡 Model ทำนายแม่นมาก

Theoretical edge = 100 contracts × (3.88 − 3.25) = 100 × 0.63 = +63.00
ผลจริง = +62.81 · ต่างกันแค่ 0.19 — พิสูจน์ว่าถ้า input ถูก (โดยเฉพาะ volatility) model ทำนายกำไรได้แม่นจริง

⚡ จุดสำคัญเรื่อง Adjustments

การ adjust บังคับให้เรา "buy low, sell high" โดยอัตโนมัติ — ราคาขึ้น delta บวก ต้องขาย futures (sell high) · ราคาลง delta ลบ ต้องซื้อ futures (buy low) · นี่คือที่มาของกำไร +205.27 ที่ชดเชย hedge เดิมที่ขาดทุน

04 มอง Hedge เป็น "การแข่งขัน"

"We might think of the hedge as a race between the loss in time value of the calls and the cash flow resulting from the adjustments, with the theoretical pricing model acting as the judge."

— Natenberg, Chapter 5
THE RACE — เมื่อซื้อ undervalued option
▼ ฝั่งที่เสีย
Time Value decay ของ option ที่ซื้อมา (theta)
VS
▲ ฝั่งที่ได้
Cash flow จาก adjustments (buy low sell high)
ใครชนะการแข่งขัน?

ถ้าซื้อ option ที่ ราคา < theoretical value → adjustments จะชนะ (กำไร) · ถ้าซื้อแพงกว่า theoretical → time decay จะชนะ (ขาดทุน) · กติกาของการแข่งขันถูกกำหนดโดย input เข้า model โดยเฉพาะ volatility

05 Volatility คือตัวตัดสิน · Break-even Vol

ในตัวอย่าง future volatility = 18.3% (ที่สมมติว่ารู้) แต่ถ้า vol จริงต่างออกไปล่ะ?

Vol สูงกว่าที่จ่าย

price ขยับมากขึ้น → adjustments บ่อยขึ้น → กำไรมากขึ้น (สอดคล้องกับ "vol สูง = option แพงขึ้น")

Vol ต่ำกว่าที่จ่าย

price ขยับน้อย → adjustments น้อย → กำไรลด · ถ้า vol ต่ำพอ กำไรเป็น 0

💡 Break-even Volatility = Implied Volatility

vol ที่ทำให้กำไรเป็น 0 พอดี = implied volatility ของราคาที่ซื้อ · ตัวอย่าง: ซื้อ call ที่ 3.25 → IV = 14.6% · ถ้า vol จริง > 14.6% = กำไร · ถ้า < 14.6% = ขาดทุน · เราซื้อเพราะคาดว่า vol จริงจะเป็น 18.3% > 14.6%

06 2 วิธีเก็บกำไร · ไม่ต้องถือถึง Expiry

ไม่จำเป็นต้องถือ position 10 สัปดาห์เต็มเสมอ — มี 2 ทางที่ edge จะกลายเป็นเงิน:

A
IV reverts ขึ้น (เร็ว)
ถ้า implied vol ในตลาดวิ่งขึ้นจาก 14.6% → 18.3% → ราคา option ขึ้นจาก 3.25 → 3.88 ทันที → ปิด position เก็บกำไร 0.63 เลย ไม่ต้องรอ
B
ถือถึง expiry + adjust (ช้า)
ถ้า IV ไม่ reverts → ถือ position แล้ว adjust ไปเรื่อยๆ จนถึง expiry → กำไรจะค่อยๆ realize ผ่าน cash flow ของ adjustments
⚑ IV อาจเคลื่อนสวนทางชั่วคราว

หลังเปิด position IV อาจตกลงไปอีก (เช่น 14.6% → 13.5%) → เกิด paper loss ทันที · แต่ถ้า vol forecast (18.3%) ยังถูก ก็ไม่ต้องปิด — เหมือน trader เลือก bottom ของ IV ไม่ได้แม่นเป๊ะ ต้องทนรับ temporary loss เป็น

07 Retail vs Professional · ใครควร adjust แค่ไหน

ประเด็นที่สำคัญมากสำหรับ retail trader — Natenberg อธิบายว่าทั้งสองมี edge เท่ากัน แต่ approach ต่างกัน:

Professional (low cost)

transaction cost ต่ำ → adjust บ่อยได้ → ลด short-term luck → กำไรนิ่งกว่า

Retail (high cost)

transaction cost สูง → adjust น้อย หรือไม่ adjust → กำไรเหวี่ยงมากกว่า แต่ expected value เท่ากัน

💡 Adjustments ไม่เพิ่ม expected return

การ adjust ไม่ได้เพิ่มกำไรคาดหวัง — มันแค่ ลด variance (ความเหวี่ยง) · retail ที่ adjust น้อยจะเจอทั้งกำไรและขาดทุนที่ใหญ่กว่า แต่ในระยะยาว average เท่ากับ professional · ดังนั้น retail ที่ cost สูงควร adjust น้อยลงเพื่อไม่ให้ fee กิน edge

08 ข้อสมมติ Frictionless Market · ที่ไม่จริงในโลกจริง

Black-Scholes สมมติตลาดไม่มี friction — แต่โลกจริงไม่ใช่:

ข้อสมมติโลกจริง
ซื้อขาย underlying ได้เสรีมี daily limit, locked market
กู้/ให้กู้ rate เดียวกันมี spread ระหว่าง borrow/lend rate
Transaction cost = 0มี brokerage fee จริง (สำคัญมากสำหรับ retail)
ไม่มีภาษีมี tax consequences ต่างกันตาม strategy
ผลกระทบต่อ retail

Transaction cost คือ friction ที่กระทบ retail มากที่สุด — ถ้า fee สูง การ adjust บ่อยอาจกิน profit หมด · ต้องชั่งน้ำหนักระหว่าง adjust (ลด variance) กับ fee (ลด profit)

⚡ CONNECT TO YOUR TRADING

Webull capital ~$2,000: ในฐานะ retail ที่ cost ต่อ trade ค่อนข้างสูงเทียบทุน — บทนี้บอกชัดว่าอย่า adjust บ่อยเกินไป เพราะ fee จะกิน edge · เลือก adjust เฉพาะตอน delta เบี่ยงมากจริงๆ

IV Rank เป็นตัวตัดสิน buy/sell: หลักการ break-even vol = IV ตรงกับที่เคยคุยเรื่อง UUUU (IV Rank ~5 = ถูก → ซื้อ LEAPS) · ถ้า IV ปัจจุบัน < vol ที่คาดว่าจะเกิดจริง → ซื้อมี edge · ถ้า IV > vol ที่คาด → ขายมี edge (CSP/CC)

CSP/Covered Call = ฝั่งขาย: ขั้นตอนที่ 1 ของบทนี้ ฝั่ง "sell overvalued" คือสิ่งที่ทำอยู่ — ขาย option ตอน IV สูง (overpriced) แล้วเก็บ premium · ความต่างคือ retail ส่วนใหญ่ไม่ delta-hedge แบบในหนังสือ แต่รับ directional risk แทน (cash-secured)

2 วิธีเก็บกำไร = exit strategy: เมื่อขาย CSP ตอน IV สูง → ถ้า IV crush ลงเร็ว (วิธี A) สามารถปิดเก็บกำไรก่อน expiry ได้เลย ไม่ต้องรอ · นี่คือเหตุผลที่หลายคนปิด CSP ที่ 50% profit แทนถือถึง expiry
